## YouTube Video Downloader and Converter App

This is a Kotlin and Jetpack Compose based Android application with following features.

## Features

- **Download YouTube Videos and Playlists:** Easily download individual videos or entire playlists from YouTube.
- **Convert Video to GIF:** Convert downloaded videos into GIF format for easy sharing.
- **Convert Video to Audio:** Extract audio from videos and save it in popular audio formats.
- **Video and Music Player:** Use the app to play both video and audio files with a built-in media player.

## Home Screen
- It allows user to play video and audio files effortlessly.
- It supports adjustable playback speed, forward and backward navigation within media.
- Utilizes the Android Exoplayer Media Library for robust and smooth playback.

## Yotube Downloader Screen
- Download YouTube videos in various file formats (video and audio) and resolutions.
- Supports downloading entire playlists.
- Powered by the Python Pytube library for efficient and reliable downloading.

## Converter Screen
- Allows user to convert video to GIF as well as audio files.
- Uses the FFMPEG Mobile Module for seamless conversion, providing live conversion statistics for real-time progress tracking.
